B) How To Change The Template - top

How to change the logo

Open the Minicon HTML folfer, go to the Assets > img > and find the image logo.png and replace with your logo image. Your logo will be replace throughout the site. In the template the size of the logo image is 281px x 90px. And you can also change the logo for mobile version too. Mobile version image name is logo-mobile.png and replace with your mobile version logo. Below you can see the code structure of the Logo HTML too.

    
    <h1>
        <a href="index.html">
            <img class="img-responsive" src="assets/img/logo.png" alt="logo"/>
            <img class="logo-responsive" src="assets/img/logo-mobile.png" alt="logo-responsive"/>
        </a>
    </h1>



How to change the Menu

You can easily find the nav tag in the html code and change the menu links according to your desisre and as well href attributes to a corresponding pages

How to change the template main colors

Go to Assets > css > style.css and open the style.css file. If you want to change the main heading black colors go to the line numbers 68, 75, 78, 1735, 1776, 2245 and change this color: #3e3e3e; to color: #YOURCOLOR;

If you want to change the Golden heading colors go to the line numbers 72, 81, 560, 1796, 2250 and change this color: #c8b370; to color: #YOURCOLOR;

If you want to change the Bullet points text colors go to the line numbers 84, 1212, 1308, 3193 and change this color: #3e3e3e; to color: #YOURCOLOR;

If you want to change the Paragraph text color go to the line number 65 and change this color: #707070; to color: #YOURCOLOR;. It will change all the paragraph text color throughout the site.

If you want to change the Button Background color go to the line number 3320 and change this color: #ebe0b2; to color: #YOURCOLOR;

If you would like to edit a specific section of the template, simply go to HTML folder Assets > CSS > style.css and then open the file style.css, There will be a [Table of content CSS] at top of the file which shows the formation of the CSS. Copy the appropriate label from the table and the find the CSS and modify.




How to change the Portfolio Item Images

Go to Assets > img > placeholder and find the specific placeholder image and replace with your images. Your image size should be 480px x 480px for portfolio items. Also you can search Portfolio images with names which will be are "portfolio-img-1.jpg", "portfolio-img-2.jpg", "portfolio-img-3.jpg" etc. The HTML structure has been shown below for the One portfolio item

    
    <!--Item 1-->
    <a href="portfolio.html"> 
        <div class="grid">
            <figure class="element-item effect-portfolio Branding">
                <img class="img-responsive" src="assets/img/placeholder/portfolio-img-1.jpg" alt="portfolio-img-1"/>
                <figcaption>
                    <h4>Branding</h4>
                </figcaption>
            </figure>
        </div>
    </a>



How to change the other template images

Go to Assets > img > placeholder and find the specific placeholder image along with the sizes and replace with your images. Your images will be replace :-)

C) CSS Files - top

The template contains 3 CSS files stored in a folder: assets\css\

  1. style.css ( Master stylesheet include all template styles )
  2. animate.min.css ( CSS Library animations )
  3. bootstrap.min.css ( Style settings for the bootstrap )

The following files in your <head> section of the each pages are CSS files. Please ensure that all of them have the correct path.

    
     <!-- ALL CSS -->
     <link href="assets/css/bootstrap.min.css" rel="stylesheet" type="text/css">
     <link href="assets/css/animate.min.css" rel="stylesheet" type="text/css">
     <link href="assets/css/style.css" rel="stylesheet" type="text/css">



D) JavaScript Files - top

The template contains 12 js files stored in a folder: \assets\js\

  1. jquery-3.1.0.min.js ( jQuery is a fast, small, and feature-rich )
  2. bootstrap.min.js ( Bootstrap included scripts )
  3. contact.js ( The script includes a custom code for Contact form and Project Form )
  4. custom-map.js ( The library includes a small scripts for the Google Map Styling )
  5. custom-scripts.js ( The library includes a small scripts for the Counter Numbers, Page Loader and Isotope Portfolio items )
  6. jquery.counterup.js ( By Deafult for Counter Numbers )
  7. isotope.pkgd.min.js ( By defult for Portfolio Items )
  8. jquery.inview.min.js ( Inview is a jQuery plugin used to add or remove CSS classes to Html elements when they enter or leave )
  9. jquery.waypoints.min.js ( Waypoints is the easiest way to trigger a function when you scroll to an element )
  10. menu.js ( The library includes a custom small scripts for the top right Menu )
  11. modernizr.js ( Modernizr tells you what HTML, CSS and JavaScript features the userâ€™s browser has to offer )

E) PHP Files - top

There are two PHP files that comes with Minicon that sends the information from the contact form to an email. The only editing you need to do in these files is edit it so it sends to the correct email.

  • sendEmail.php ( Contact Form Send File )
  • sendProjectdetailEmail.php ( Project Inquiry Form Send File )

Open sendEmail.php and find $site_owners_email = 'example@email.com'; and change example@email.com to your email. Then, save the file and your contact form will be functional. Keep in mind that for the PHP to work, the site must be uploaded to a webserver with PHP capabilities, you cannot run it locally on your computer, if you don't have PHP capabilities please contact your hosting support.

And you need to repeat this above steps for this file sendProjectdetailEmail.php too to make the Project Inquiry form functional.
